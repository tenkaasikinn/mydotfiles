# Dotfiles

Personal Linux/MacOS dotfiles used for daily development and DevOps work.

This repository contains configuration files that I personally use.
Some files are used as-is from upstream projects.

## Inside
- Zsh configuration
- Alacritty configuration
- Fastfetch theme
- Oh My Posh theme
